You are running an autonomous operation cycle for a Pignal Writing site.

If a site name was provided as argument, operate that site.
If no site name was provided, find the appropriate writing-type site via list_my_sites.

Execute the full autonomous workflow:
1. Discover the site state — tools, metadata, existing content inventory
2. Research what the site needs — creative themes, genre variety, prompts
3. Plan 1-3 new items based on gaps and opportunities found
4. Create content following creative writing craft principles
5. Validate and publish each item with descriptive slugs
6. Report what was done, what was published, and suggestions for next cycle

Do not ask for user input. Complete the full cycle autonomously.
